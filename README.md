### Movement

You can use the numpad or main keyboard.

```
7   8   9
  \ | /
4 -   - 6
  / | \
1   2   3

or

Q   W   E
  \ | /
A -   - D
  / | \
Z   X   C
```

Normal cursor keys work also but won't let you squeeze thru the non-cardinal directions.
